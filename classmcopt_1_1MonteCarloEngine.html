Detailed Description

High-performance parallel Monte Carlo pricing engine.

This class simulates the paths of the underlying asset using Geometric Brownian Motion (GBM) under the Risk-Neutral measure:

\[ dS_t = r S_t dt + \sigma S_t dW_t \]

Key Features:

  • Parallel Execution: Uses std::async and std::future for multi-threaded simulation.
  • Variance Reduction: Implements Antithetic Variates technique (using \( Z \) and \( -Z \)).
  • Reproducibility: Supports deterministic execution via seeded RNG (std::mt19937_64).
  • Greeks Calculation: Computes Delta and Gamma using Finite Difference Methods.

Greeks mcopt::MonteCarloEngine::calculateGreeks ( unsigned long long  numSimulations)

Calculates Price, Delta, and Gamma simultaneously.

Uses Finite Difference Method (FDM) with a small bump \( h \):

  • Delta: Central difference \( \Delta \approx \frac{V(S+h) - V(S-h)}{2h} \)
  • Gamma: Second order difference \( \Gamma \approx \frac{V(S+h) - 2V(S) + V(S-h)}{h^2} \)
Parameters
numSimulationsTotal number of paths to simulate for each spot point.
Returns
A Greeks structure containing price, delta, and gamma.
