Detailed Description

Abstract base class for option payoff strategies.

Implements the Strategy Design Pattern. This interface allows the Monte Carlo engine to be decoupled from the specific option type (Call, Put, etc.). Derived classes must implement the operator() to calculate the payoff based on the terminal spot price.
