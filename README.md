# Parallel Monte Carlo Option Pricing

Программная реализация численного метода Монте-Карло для оценки стоимости европейских опционов и расчета греков (Delta, Gamma).
Проект выполнен в рамках курса «Программная инженерия и C++ для количественного анализа» (МГУ, Мехмат).

## Описание задачи

Реализован движок (pricing engine), моделирующий эволюцию цены базового актива как геометрическое броуновское движение (модель Блэка-Шоулза-Мертона). 

Ключевые особенности реализации:
1. **Параллельные вычисления:** Использование `std::async` для распараллеливания симуляций.
2. **Понижение дисперсии:** Метод антитетических переменных (Antithetic Variates) для ускорения сходимости.
3. **Генерация случайных чисел:** Использование 64-битного Вихря Мерсенна (`std::mt19937_64`) с локальным состоянием потока (`thread_local`) для исключения гонки данных.
4. **Расчет греков:** Численное дифференцирование (метод конечных разностей) для оценки Delta и Gamma.
5. **Верификация:** Автоматическая сверка результатов с аналитическим решением Блэка-Шоулза.

## Требования к сборке

* Компилятор с поддержкой C++17 (GCC 7+, Clang 6+, MSVC 2017+).
* CMake версии 3.14 или выше (для поддержки FetchContent).
* GoogleTest (загружается автоматически через CMake).

## Инструкция по сборке и запуску

Проект собирается стандартными средствами CMake. Рекомендуется сборка в конфигурации Release для включения оптимизаций.

Linux / macOS:
```bash
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
./OptionPricer
```

Windows (PowerShell):
```bash
mkdir build
cd build
cmake ..
cmake --build . --config Release
.\Release\OptionPricer.exe
```

## Тестирование
Для запуска unit-тестов (проверка паритета пут-колл, сходимость Монте-Карло, точность аналитической формулы):
```bash
cd build
ctest --output-on-failure
```
